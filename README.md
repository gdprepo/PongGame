
# PongGame
L'objectif de ce projet est de programmer le jeu d'arcade *Pong* en Python

![](https://upload.wikimedia.org/wikipedia/commons/thumb/f/f8/Pong.png/280px-Pong.png)

## Outils utilisés
* Python3
* Tkinter

## Commandes
- **Raquette de gauche :**
    * Aller en Haut : `R`
    * Aller en Bas : `F`
- **Raquette de droite :**
    * Aller en Haut : `↑`
    * Aller en Bas : `↓`
